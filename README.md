# Youtube-Playlist-Downloader
Utilizing python yt-dlp to make it easier to download youtube playlists. Essential for archiving youtube channels and downloading songs to load onto your phone.
# Before Running 
```pip install yt-dlp ```
# The Script will make a directory or you can make your own
- the Default Directory name is ```YouTube_Downloads```
# To Use
- Edit the array and put the playlists in there.
